# BLUEPINK
2021 Konkuk AI Hackathon Project

## 서비스 소개
- 심리 질의응답을 통한 성격 분류와 감정 분류 수행
- AI Hub의 한국어 감정 대화를 학습시킨 챗봇을 이용한 감정 심리 상담
- 성격과 감정에 기반한 분류화된 상담
- 사용자의 우울 정도에 따른 맞춤 조언 / 해결 방안 제공
